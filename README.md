# Информационная система наблюдения за посетителями пляжа

**Проект:** Система учёта посещений пляжа и использования инвентаря

##  Описание проекта

Информационная система предназначена для учёта посетителей пляжа и отслеживания использования пляжного инвентаря (шезлонги, плавательные жилеты/круги, надувные матрасы). Система позволяет фиксировать время прихода и ухода посетителей, отметки об использовании инвентаря, а также формировать детализированные отчёты для менеджера.

## Функциональные требования (базовый CRUD)

### Сущности системы:

#### 1. Посетитель (Visitor)
| Поле | Тип | Описание |
|------|-----|----------|
| id | Integer | Уникальный идентификатор |
| zone_id | Integer | ID пляжной зоны (внешний ключ) |
| arrival_time | DateTime | Время прихода |
| departure_time | DateTime | Время ухода |
| used_sunbed | Boolean | Использование шезлонга (Ш) |
| used_float | Boolean | Использование плавсредства (П) |
| used_mattress | Boolean | Использование матраса (М) |
| notes | Text | Дополнительные заметки |

#### 2. Пляжная зона (BeachZone)
| Поле | Тип | Описание |
|------|-----|----------|
| id | Integer | Уникальный идентификатор |
| name | String(100) | Название зоны |
| description | Text | Описание местоположения |
| polygon_data | JSON | Координаты границ (полигон) |
| center_lat | Float | Центр зоны (широта) |
| center_lng | Float | Центр зоны (долгота) |
| color | String(20) | Цвет маркера на карте |

#### 3. Отчёт (Report)
| Поле | Тип | Описание |
|------|-----|----------|
| id | Integer | Уникальный идентификатор |
| zone_id | Integer | ID пляжной зоны |
| start_date | Date | Начало периода |
| end_date | Date | Конец периода |
| generated_by | Integer | ID сотрудника |
| generated_at | DateTime | Дата формирования |
| report_data | JSON | Данные отчёта |
| google_doc_link | String(500) | Ссылка на Google Doc |

### Основные операции CRUD:

####  Посетители
- **Create**: Добавление нового посетителя (фиксация времени прихода)
- **Read**: Просмотр списка посетителей с фильтрацией по дате и зоне
- **Update**: Редактирование времени ухода и отметок об использовании инвентаря
- **Delete**: Удаление записи о посетителе (при необходимости)

#### Пляжные зоны
- **Create**: Добавление новой пляжной зоны с полигоном на карте
- **Read**: Просмотр списка зон и их расположения на карте
- **Update**: Редактирование информации о зоне
- **Delete**: Удаление зоны

####  Отчёты
- **Create**: Генерация нового отчёта за выбранный период
- **Read**: Просмотр списка сгенерированных отчётов
- **Delete**: Удаление старых отчётов

##  Форма отчёта для менеджера

Менеджер выбирает детскую площадку (пляжную зону) и период (дата начала, дата окончания). Отчёт формируется в виде таблицы:

| Дата | 08:00-12:00 | 12:00-16:00 | 16:00-20:00 |
|------|-------------|-------------|-------------|
|      | Ш | П | М | Ш | П | М | Ш | П | М |
|------|---|---|---|---|---|---|---|---|---|
| 01.06.2026 | 5 | 2 | 3 | 8 | 4 | 5 | 6 | 3 | 2 |
| 02.06.2026 | 4 | 1 | 2 | 7 | 3 | 4 | 5 | 2 | 3 |

*Ш - шезлонги, П - плавсредства (жилеты/круги), М - матрасы*

В столбцах отображается **общее количество посетителей**, пользовавшихся предметом из соответствующей категории в указанный временной промежуток.
